# Dart Studying

Este repositório foi criado para documentar e organizar meus estudos, exercícios e projetos práticos na linguagem **Dart**.

---

## 🚀 Objetivo do Projeto
O foco principal é dominar a sintaxe da linguagem, entender os conceitos de Programação Orientada a Objetos (POO) e praticar a lógica de programação antes de avançar para o desenvolvimento de aplicativos mais robustos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Dart
* **Ambiente de Desenvolvimento:** Linux (Terminal / Geany)

## 📁 Estrutura do Repositório
* `/conceitos_basicos`: Variáveis, tipos de dados, estruturas condicionais e loops.
* `/funcoes`: Criação de funções, parâmetros e escopos.
* `/poo`: Classes, objetos, herança, polimorfismo e abstração.
* `/desafios`: Pequenos algoritmos e problemas resolvidos para fixação.

## 🏃 Como Executar os Arquivos
Para rodar qualquer script Dart deste repositório usando o terminal Linux, certifique-se de ter o Dart SDK instalado e execute:
